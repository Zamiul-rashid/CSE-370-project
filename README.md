# Football Analyzer

A Django app for exploring football formations, tactics, managers, match events, and simple tactical analytics.

## Setup

1. Create a virtual environment.
2. Install dependencies with `pip install -r requirements.txt`.
3. Run `python manage.py runserver`.
4. Open `http://127.0.0.1:8000/`.

## Data

The project uses SQLite with sample football data already loaded in `db.sqlite3`.

## Main pages

- Home dashboard
- Formation comparison
- Tactical timeline
- Formation builder
- Manager profiling
- Tactical suggestion
- Opponent adaptation
- Player role evolution
- Heatmap analysis
- Tactical efficiency
